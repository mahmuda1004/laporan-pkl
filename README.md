## Deskripsi Proyek
Proyek ini merupakan hasil Praktik Kerja Lapangan (PKL) yang bertujuan untuk membuat **website aplikasi** sebagai media laporan (report) kegiatan Sales Agent pada Divisi Business Service di Telkom STO Dahlia Kota Samarinda.  

Website ini memudahkan sales agent untuk:  
- Melaporkan aktivitas canvasing dan penjualan.  
- Mengelola data pelanggan dan tagihan.  
- Mencatat biaya operasional seperti bahan bakar kendaraan karyawan.  
- Mengarsipkan dokumentasi kegiatan berupa foto dan laporan.  

Selama PKL, penulis juga melakukan kegiatan pendukung seperti survei lokasi pemasangan Wi-Fi UMK, menelpon pelanggan secara berkala, dan mengedit dokumentasi kegiatan sales.

## Teknologi yang Digunakan
- **HTML & CSS** – untuk struktur dan desain website.  
- **PHP** – untuk pengolahan data dan interaksi dengan database.  
- **MySQL** – sebagai database untuk menyimpan data laporan dan pengguna.  
- **XAMPP** – sebagai server lokal untuk pengembangan website.  
- **Web Browser (Chrome/Edge)** – untuk menampilkan dan menguji website.

## Fitur Website
1. **Sign Up & Log In** – sistem autentikasi untuk Sales Agent.  
2. **Dashboard** – menampilkan ringkasan kegiatan dan laporan.  
3. **Input Data** – mengisi laporan aktivitas harian dan biaya operasional.  
4. **My Account & Change Password** – pengaturan akun pengguna.  
5. **About Us** – informasi mengenai website dan tujuan pembuatan.  
6. **Dokumentasi Foto** – upload dan tampilan dokumentasi kegiatan sales.

## Cara Menjalankan Website
1. Install **XAMPP** atau server lokal lainnya.  
2. Import file `db.sql` ke **MySQL** melalui phpMyAdmin.  
3. Salin seluruh folder proyek ke folder `htdocs` (jika menggunakan XAMPP).  
4. Jalankan **Apache** dan **MySQL** melalui XAMPP Control Panel.  
5. Akses website melalui browser dengan alamat: `http://localhost/[nama-folder-proyek]`.
